Creating Websites On Github Using HTML

  1. Create a respository. The name of this respository must be your username+"github.oi". For example, if your username is pamelafoxbot you must name your respository "pamelafoxbot.github.oi".

  2. Creating a file and name it "index.html". This file will be your homepage.

  3. (Optional) Add other files to your respository to create more webpages. The url of these webpages are always your respository's webpage url+the file name. For example, the url of this website is "https://mathcodeforfun.github.io/" and the file of this webpage is"Github-Sites.html" so the url of this webpage is "https://mathcodeforfun.github.io/MathCodeForFun.github.oi/Github-Sites.html".

  4. Go to the settings of your respository and under the section of the sidebar that says "Code and automation", their is a button that says "Pages". Click on that button.

  5. Under the bold text that says "Source", select "Deploy from a branch".

  6. Under the bold text that says "Branch", use the dropdown menu to select a branch, then another dropdown menu will appear. You can select "/docs" or "/root".

  7. Press "Save".

  8. Go back to your respository. Below the button that says "Go to file", their will be a small, brown dot. When this dot is on your screen that means that your website is being created or updated. Wait until this dot becomes a green checkmark. If the dot becomes a red x, you must fix your website before you can visit it.

  9. Finally, once the green checkmark appears, you can visit your new website!
